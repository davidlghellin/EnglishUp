# Modal Verbs

## Can, Could, Should, Must, Have To, Would Like To

---

## What are Modal Verbs?

Modal verbs are special verbs that add meaning to the main verb. They express:
- Ability (can, could)
- Permission (can, could)
- Obligation (must, have to)
- Advice (should)
- Wishes/Desires (would like to)

```
Modal + base verb (no TO, no -S, no -ING)
```

---

## CAN - Ability & Permission

### Form

```
Subject + CAN + verb (base form)
Subject + CAN'T + verb (base form)
CAN + subject + verb?
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **Ability** | I **can** speak English | Puedo/Sé hablar inglés |
| **Permission** | You **can** go now | Puedes irte ahora |
| **Possibility** | It **can** be difficult | Puede ser difícil |
| **Requests** | **Can** you help me? | ¿Puedes ayudarme? |

### Examples

| Affirmative | Negative | Question |
|-------------|----------|----------|
| I **can** code in Python | I **can't** code in Java | **Can** you code? |
| She **can** drive | She **can't** swim | **Can** she come? |

---

## COULD - Past Ability & Polite Requests

### Form

```
Subject + COULD + verb (base form)
Subject + COULDN'T + verb (base form)
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **Past ability** | I **could** run fast when I was young | Podía correr rápido |
| **Polite request** | **Could** you repeat that? | ¿Podría repetir eso? |
| **Possibility** | It **could** rain tomorrow | Podría llover mañana |
| **Suggestion** | We **could** try a different approach | Podríamos probar otro enfoque |

### Can vs Could (Politeness)

| Less formal | More formal/polite |
|-------------|-------------------|
| **Can** you help me? | **Could** you help me? |
| **Can** I use your phone? | **Could** I use your phone? |

---

## SHOULD - Advice & Recommendations

### Form

```
Subject + SHOULD + verb (base form)
Subject + SHOULDN'T + verb (base form)
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **Advice** | You **should** rest more | Deberías descansar más |
| **Recommendation** | We **should** test this code | Deberíamos probar este código |
| **Opinion** | They **should** fix that bug | Deberían arreglar ese bug |

### Examples

| Affirmative | Negative |
|-------------|----------|
| You **should** learn Git | You **shouldn't** work so much |
| We **should** update the server | We **shouldn't** ignore the warnings |
| She **should** take a break | She **shouldn't** skip meals |

---

## MUST - Strong Obligation & Certainty

### Form

```
Subject + MUST + verb (base form)
Subject + MUSTN'T + verb (base form)
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **Strong obligation** | You **must** finish this today | Debes terminar esto hoy |
| **Prohibition** | You **mustn't** share passwords | No debes compartir contraseñas |
| **Certainty** | He **must** be tired | Debe de estar cansado |

### Examples

| Obligation | Prohibition |
|------------|-------------|
| You **must** attend the meeting | You **mustn't** be late |
| We **must** backup the data | You **mustn't** delete production files |

---

## HAVE TO - Obligation (External)

### Form

```
Subject + HAVE TO / HAS TO + verb
Subject + DON'T HAVE TO / DOESN'T HAVE TO + verb
DO/DOES + subject + HAVE TO + verb?
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **External obligation** | I **have to** work 76 hours a week | Tengo que trabajar 76 horas |
| **Rules/Requirements** | You **have to** have a visa | Tienes que tener un visado |
| **Necessity** | We **have to** deploy by Friday | Tenemos que desplegar el viernes |

### Must vs Have To

| Must (internal/personal) | Have to (external/rules) |
|--------------------------|-------------------------|
| I **must** exercise more | I **have to** exercise (doctor's orders) |
| (I feel I need to) | (Someone told me to) |

### Don't Have To vs Mustn't

| Don't have to (no obligation) | Mustn't (prohibited) |
|-------------------------------|----------------------|
| You **don't have to** come | You **mustn't** come |
| (It's optional) | (It's forbidden) |

### Examples with "Have To"

| Affirmative | Negative | Question |
|-------------|----------|----------|
| I **have to** work late | I **don't have to** work weekends | **Do** you **have to** travel? |
| She **has to** finish the report | She **doesn't have to** attend | **Does** she **have to** sign? |

---

## WOULD LIKE TO - Wishes & Polite Requests

### Form

```
Subject + WOULD LIKE TO + verb (base form)
Subject + WOULDN'T LIKE TO + verb
WOULD + subject + LIKE TO + verb?
```

### Uses

| Use | Example | Spanish |
|-----|---------|---------|
| **Wishes** | I **would like to** live abroad | Me gustaría vivir en el extranjero |
| **Polite desires** | I **would like to** order coffee | Quisiera pedir un café |
| **Hypothetical** | I **would like to** live in a country with lower taxes | Me gustaría vivir en un país con impuestos más bajos |

### Want vs Would Like

| Want (direct) | Would like (polite) |
|---------------|---------------------|
| I **want** a coffee | I **would like** a coffee |
| I **want to** go home | I **would like to** go home |

### Examples

| Example | Context |
|---------|---------|
| I **would like to** change careers | Expressing a wish |
| **Would** you **like to** join the meeting? | Polite invitation |
| She **would like to** learn Python | Desire |
| We **wouldn't like to** delay the project | Negative preference |

### Contractions

- I would = I'd
- You would = You'd
- He/She would = He'd/She'd
- We would = We'd
- They would = They'd

| Full form | Contracted |
|-----------|------------|
| I **would like to** go | I**'d like to** go |
| She **would like to** help | She**'d like to** help |

## Summary Table

| Modal | Use | Example |
|-------|-----|---------|
| **can** | ability, permission | I can code in Python |
| **could** | past ability, polite request | Could you help me? |
| **should** | advice | You should rest |
| **must** | strong obligation | You must attend |
| **have to** | external obligation | I have to work 76 hours |
| **would like to** | wishes, polite desires | I'd like to live abroad |

---

## Common Mistakes

| Wrong | Correct | Explanation |
|-------|---------|-------------|
| I ~~can to~~ swim | I **can** swim | No "to" after can |
| She ~~musts~~ go | She **must** go | No -s on modals |
| He ~~cans~~ drive | He **can** drive | No -s on modals |
| I ~~must to~~ work | I **must** work | No "to" after must |
| I ~~don't must~~ | I **mustn't** / I **don't have to** | Different meanings |
| She ~~would likes~~ | She **would like** | No -s after would |
| I ~~have to going~~ | I **have to go** | Base verb after have to |

## Practice Exercises

### Exercise 1: Choose the correct modal

1. I _______ speak three languages. (ability)
2. You _______ be more careful with passwords. (advice)
3. We _______ finish the project by Monday. (external obligation)
4. _______ you repeat that, please? (polite request)
5. I _______ live in a warmer country. (wish)

<details>
<summary>Answers</summary>

1. can
2. should
3. have to
4. Could
5. would like to
</details>

### Exercise 2: Correct the mistakes

1. She cans speak French.
2. I must to finish this today.
3. You don't must smoke here.
4. I would likes to help you.
5. He have to work on Saturdays.

<summary>Answers</summary>

2. I **must** finish this today.
