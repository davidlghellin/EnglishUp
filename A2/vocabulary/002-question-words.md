# Question Words - Palabras Interrogativas

## Main Question Words

| Español | English | Uso / Use |
|:-------:|:-------:|:----------|
| ¿Qué? | What? | Para preguntar sobre cosas / To ask about things |
| ¿Quién? | Who? | Para preguntar sobre personas / To ask about people |
| ¿Dónde? | Where? | Para preguntar sobre lugares / To ask about places |
| ¿Cuándo? | When? | Para preguntar sobre tiempo / To ask about time |
| ¿Por qué? | Why? | Para preguntar sobre razones / To ask about reasons |
| ¿Cómo? | How? | Para preguntar sobre manera / To ask about manner |
| ¿Cuál/Cuáles? | Which? | Para elegir entre opciones / To choose between options |
| ¿De quién? | Whose? | Para preguntar sobre posesión / To ask about possession |

## Question Words + Other Words

| Español | English | Ejemplo / Example |
|:-------:|:-------:|:------------------|
| ¿Cuántos/as? | How many? | **How many** books do you have? |
| ¿Cuánto? | How much? | **How much** does it cost? |
| ¿Qué tan...? | How + adjective? | **How** old are you? / **How** tall is he? |
| ¿A qué hora? | What time? | **What time** do you start work? |
| ¿Qué tipo de...? | What kind of...? | **What kind of** music do you like? |
| ¿De dónde? | Where... from? | **Where** are you **from**? |

## Examples and Usage

### What - ¿Qué?
- **What** is your name?
- **What** do you do? (job)
- **What** are you doing?
- **What** time is it?

### Who - ¿Quién?
- **Who** is that person?
- **Who** are you talking to?
- **Who** called me?

### Where - ¿Dónde?
- **Where** do you live?
- **Where** is the bank?
- **Where** are you going?
- **Where** are you **from**?

### When - ¿Cuándo?
- **When** is your birthday?
- **When** did you arrive?
- **When** does the movie start?

### Why - ¿Por qué?
- **Why** are you late?
- **Why** did you call me?
- **Why** is the shop closed?

**Answer with "Because..."**: Because I missed the bus.

### How - ¿Cómo?
- **How** are you?
- **How** do you spell that?
- **How** did you get here?
- **How** old are you?
- **How** tall is she?
- **How** far is it?

### Which - ¿Cuál?
- **Which** color do you prefer, blue or red?
- **Which** bus goes to the center?
- **Which** one do you want?

### Whose - ¿De quién?
- **Whose** bag is this?
- **Whose** car is parked outside?

## Common Question Structures

| Structure | Example |
|:----------|:--------|
| Question word + **do/does** + subject + verb? | What **do** you like? |
| Question word + **did** + subject + verb? | Where **did** she go? |
| Question word + **is/are** + subject? | Who **is** that? |
| Question word + **was/were** + subject? | Where **were** you yesterday? |
| Question word + **will** + subject + verb? | When **will** you arrive? |
| Question word + **can** + subject + verb? | How **can** I help you? |
